# 서비스 개요

AI 도구 사용량을 로컬 우선으로 수집/정리하고, 사용자에게 비용/생산성 의사결정을 돕는 인사이트를 제공한다.

## 현재 제품 상태 (2026-02-18)
- Agent + 로컬 SQLite + Windows Tkinter 대시보드의 MVP 구현은 완료되었다.
- OAuth 로그인, 일/주 집계 조회, 예산 임계치 알림 UI가 동작한다.
- 자동화 테스트는 로컬 단위 테스트 기준 `9 passed`로 확인되었다.

## 현재 공백
- Windows 실기기(Win10/Win11)에서 설치부터 OAuth 로그인까지의 E2E 증적이 부족하다.
- 해상도/권한/네트워크 상태별 회귀 검증 루틴이 문서화되어 있지 않다.

## 이번 루프 초점
- 테스트 전략을 Big Picture 하위 문서로 분리해 게이트 기준을 고정한다.
- Windows 실기기 검증 시나리오와 증적 포맷(로그/스크린샷)을 표준화한다.
